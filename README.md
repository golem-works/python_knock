# 🐍 Python100本基礎ノック（by golem-works）

このリポジトリは、Pythonを使って実践的なスキルを積み上げるための「ノック形式」トレーニング集です。  
構文、ファイル操作、ネットワーク通信、DB接続など、**基礎から応用まで“手を動かして”学ぶ**ことを重視しています。

> 💬 "読むだけではなく、組んで動かして壊してみる" ── それが、このノックの哲学です。

---

## 📂 構成（抜粋）

| ファイル名 | 内容概要 |
|------------|----------|
| `00_read_file.py` | 指定したファイルを読み込んでWindowsで開くファイルIOの基本 |
| `01_if_.py` | if文を用いたクイズ形式の入力チェック |
| `02_soket_remot_server.py` | socket通信によるリモートサーバープログラム（実験用）※悪用厳禁 |
| `03_client.py` | サーバーに接続するクライアント側コード（※上記とペアで使用） |
| `04_network_node.py` | networkxとmatplotlibによるノード/エッジの通信と可視化 |
| `05_hex.py` | 16進数の文字列をデコードして人間の文字列に変換する |
| `06_sql_login.py` | MySQLと接続し、ユーザー名とパスワードのログイン認証を行う |
| `07_chat_bot.py` |  辞書応答形式の簡単な応答をするchatbot |

## 🔧 使用環境

- Python 3.10 以上を推奨
- 以下の外部ライブラリを必要とするコードがあります：

```bash
pip install mysql-connector-python
pip install networkx
pip install matplotlib
````
